# Змейка (Snake Game)

## Описание проекта

Это классическая игра "Змейка", реализованная на Python с использованием библиотеки Pygame. Игрок управляет змейкой, которая движется по игровому полю, поедая яблоки и увеличиваясь в длину. Цель игры — набрать как можно больше очков, избегая столкновения с собственным телом. Если змейка сталкивается сама с собой, игра сбрасывается, и змейка возвращается в начальное состояние.

Игра включает в себя:
- Графическое отображение на экране с сеткой.
- Управление змейкой с помощью клавиш стрелок.
- Случайное появление яблок на поле.
- Проверку на столкновения.

Проект разработан с учетом принципов объектно-ориентированного программирования (ООП). Основные классы:
- `GameObject`: Базовый класс для всех игровых объектов, содержащий общие атрибуты и методы.
- `Snake`: Класс змейки, наследующий от `GameObject`. Управляет движением, ростом и проверкой столкновений.
- `Apple`: Класс яблока, наследующий от `GameObject`. Отвечает за генерацию случайных позиций.

## Структура проекта

- `the_snake.py`: Основной файл с кодом игры. Содержит классы `GameObject`, `Snake`, `Apple`, функции `handle_keys` и `main`, а также константы и инициализацию Pygame.
- `conftest.py`: Конфигурационный файл для pytest. Настраивает тестовое окружение, включая фикстуры для объектов и обработку таймаутов.
- `test_code_structure.py`: Тесты на структуру кода. Проверяют наличие необходимых атрибутов, методов и наследований в классах.
- `test_main.py`: Тесты на запуск основной функции `main`. Проверяют, что игра запускается без исключений.
- `requirements.txt`: Файл с зависимостями проекта.

## Установка

1. Убедитесь, что у вас установлен Python (версия 3.6 или выше).
2. Клонируйте репозиторий или скачайте файлы проекта.
3. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```
   Это установит необходимые пакеты: Pygame, pytest, flake8 и другие.

## Запуск игры

Чтобы запустить игру, выполните в терминале:
```
python the_snake.py
```

Игровое окно откроется, и вы сможете управлять змейкой с помощью клавиш стрелок:
- Стрелка вверх: движение вверх.
- Стрелка вниз: движение вниз.
- Стрелка влево: движение влево.
- Стрелка вправо: движение вправо.

Игра продолжается до тех пор, пока вы не закроете окно. При столкновении змейки с собой она сбрасывается.

## Тестирование

Проект включает автоматизированные тесты с использованием pytest. Тесты проверяют:
- Наличие необходимых атрибутов и методов в классах.
- Корректность наследования.
- Запуск игры без ошибок.

Чтобы запустить тесты:
```
pytest
```

Тесты используют фикстуры из `conftest.py` для создания объектов без отображения графического интерфейса (SDL_VIDEODRIVER установлен в 'dummy').

## Зависимости

- `pygame==2.5.2`: Для создания графического интерфейса и управления игрой.
- `pytest==7.1.3`: Для запуска тестов.
- `pytest-timeout==2.1.0`: Для ограничения времени выполнения тестов.
- `flake8==5.0.4`, `flake8-docstrings==1.7.0`, `pep8-naming==0.13.3`, `pycodestyle==2.9.1`: Для проверки стиля кода (PEP 8).

## Технические детали

- **Размеры поля**: 640x480 пикселей, сетка 20x20 пикселей.
- **Скорость**: 10 кадров в секунду.
- **Цвета**:
  - Фон: Черный (0, 0, 0).
  - Граница ячеек: Голубой (93, 216, 228).
  - Змейка: Зеленый (0, 255, 0).
  - Яблоко: Красный (255, 0, 0).
- **Направления движения**: Определены как кортежи (dx, dy): UP=(0, -1), DOWN=(0, 1), LEFT=(-1, 0), RIGHT=(1, 0).
- **Логика движения**: Змейка перемещается по модулю ширины/высоты экрана, создавая эффект "зацикленного" поля.
- **Сброс игры**: При столкновении змейка возвращается в центр, длина сбрасывается до 1.
